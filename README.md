# Анализ транзакционных данных (SQL)

## Описание
Проект по решению задач анализа транзакционных данных с использованием SQL. Основной фокус — работа с выборками и аналитическими запросами.

## Что сделано
- написание SQL-запросов для анализа данных  
- работа с агрегатами и группировками  
- обработка транзакционных выборок  
- оптимизация логики запросов  

## Инструменты
SQL (MySQL / PostgreSQL)

## Результат
- реализованы решения типовых аналитических задач  
- подготовлены выборки данных для анализа  
- отработаны подходы к работе с транзакционными данными
