# Анализ транзакционных данных (SQL, MySQL)

## Описание
Анализ транзакционных данных с использованием SQL. Фокус на обработке выборок и построении аналитических запросов.

## Что сделано
- написание SQL-запросов для анализа транзакций  
- работа с JOIN, агрегатами и группировками  
- формирование выборок данных  
- анализ операций и пользовательской активности  

## Инструменты
SQL (MySQL)

## Результат
- реализованы аналитические запросы для транзакционных данных  
- подготовлены выборки для анализа  
- отработаны подходы к работе с финансовыми данными
